## ğŸ›  How It Works

### 1. Shortcut Execution  
The user runs the **Create RSS for YouTube Search** Shortcut, which:
- Collects a YouTube video list for they search query
- Calls the YouTube Data API
- Builds an RSS-formatted XML string
- Saves the resulting feed inside `feed/` as a `.rss` file, using a slugified version of the search term as the filename.

### 2. Shell Script Automation (macOS)  
After saving the RSS file, the Shortcut triggers a **Run Shell Script** action that:

1. Iterates through every file inside `feed/`
2. Reads index.html as a template and replaces the section between:
```html
<!-- LINKS START -->
<!-- LINKS END -->
```
with a new list of `<li><a>` links for every `.rss` file in the `feed/` directory.
**`OBS:`** Before enter the running script, the pipeline switch blank spaces by hiphens in search input and switch special characters for your related simple version.
3. Capitalizes the string for each word.
4. Appends each entry as an HTML link

Example output link:

```html
<a href="./feed/lofi-music.rss">Lofi Music</a>
```

### 3. Optional GitHub Deployment

If enabled, the script will also:

```bash
git add .
git commit -m "{{ dinamic search string }} inserted"
git push origin main
```

This allows seamless publishing of new/updated feeds to GitHub â€” ideal for GitHub Pages.

## ğŸ“¦ Installation & Setup

## Requirements

* macOS/iPhone/iPad with `Apple Shortcuts Application`

* YouTube Data API Key

* Git (optional for CI/CD and GitHub Pages)

* GitHub repository (optional for hosting the site)

### Steps

1. Clone this repository

```bash
git clone https://github.com/ju-c-lopes/youtube-search-rss-pipeline.git .
```

2. Import the Shortcut file (.shortcut)

3. Configure your YouTube API key, available in [Google Cloud](https://console.cloud.google.com/), inside the Shortcut, where there is a placeholder for `[[your-youtube-api-key]]`

4. Ensure the feed/ directory exists

5. (Optional) Enable Git automation in the shell script, where you can remove it if you desire it.

6. Run the Shortcut to generate your first feed
